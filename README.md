# Agentic Development Environment

> An agentic development environment for all YOLO needs

## WHAT?

A ready-to-go Docker Compose configuration suitable for most projects, with instructions to make your agent use the container for everything, leaving your machine intact.

## WHY?

Because I don't like others running and messing with things on my machine; isolating agents creates an extra level of security.

## HOW?

Clone this repo, remove the `.git` dir and `README.md`, open your agent and enjoy the good vibes!
